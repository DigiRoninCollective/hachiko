## Color Palette

- RGB: 30, 58, 138
- Usage: Primary brand color, backgrounds, major UI elements
- Psychology: Trust, reliability, loyalty (74% association)

**Gold (Premium & Value)**
- Hex: #F59E0B
- RGB: 245, 158, 11
- Usage: Accent colors, CTAs, highlights, premium features

## Usage Guidelines

### Color Ratios
- Primary Blue: 60%
- Supporting Cream: 25%
- Gold Accents: 10%
- Brown Text: 5%

### Do's
- Use blue as primary brand color
- Reserve gold for premium/CTA elements
- Maintain high contrast for readability
- Use brown for authentic, grounding elements

### Don'ts
- Never use red (negative cultural associations)
- Avoid neon or overly bright variations
- Don't overwhelm with gold (maintain premium feel)
- Never use blue text on dark backgrounds

## Cultural Considerations

### Global Appeal
- Blue: Universal trust color
- Gold: Recognized value symbol globally
- Brown: Positive earth associations
- Avoids red (negative in some cultures)

### Crypto Market Positioning
- Trustworthy: Blue builds confidence
- Premium: Gold suggests value
- Different: Brown adds authenticity vs cold competitors
- Story-driven: Colors tell Hachiko's loyalty narrative
